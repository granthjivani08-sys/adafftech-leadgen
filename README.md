## About This Project

This web application was built as the practical deliverable for my 15-day Affiliate Marketing internship at **AdaffTech Pte Ltd**, submitted to **Gujarat Technological University (GTU)** as part of the Two-Week Skill Based Training Program.

It demonstrates real-world knowledge of:
- Lead generation funnels and affiliate marketing workflows
- Customer-company bridge building and client communication
- CRM principles and lead pipeline management
- Campaign performance tracking and data reporting

---

## What's Inside

### 🌐 Landing Page
- Professional lead capture form (name, email, phone, industry, goal)
- Live lead counter showing real-time stats
- Trust signals and "How It Works" process walkthrough
- Form submissions feed directly into the CRM

### 📋 CRM (Lead Manager)
- Full lead table with search, filter by status and industry
- Update lead status (New → Contacted → Qualified → Converted → Lost)
- Add, edit, and delete leads
- Manual lead entry via modal form

### 📊 Dashboard
- Live stat cards: Total Leads, Contacted, Converted, Conversion Rate
- Leads-by-day bar chart
- Pipeline breakdown with visual progress bars
- Leads-by-industry breakdown
- Recent activity feed
